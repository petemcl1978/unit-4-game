Romancing the Stone



Welcome to Romancing the Stone! In this game you will be given a random number.You will need to match that number by collecting Jewels. There are 4 jewels to select, each jewel has a value. You will need to collect the correct amount of jewels to match the number given. If you match the numbers you win the game! If you exceed the number you will loose. Good luck!
